# 📊 Rapport de Ventes Entreprise - Analyse Power BI

## 🎯 Présentation du Projet
Ce projet consiste en la création d'un tableau de bord interactif sous **Power BI** permettant d'analyser les performances commerciales d'une entreprise. L'objectif est de transformer des données brutes en indicateurs visuels pour faciliter la prise de décision.

## 📂 Structure du Dépôt
- `Formation Power BI 2025/` : Dossier contenant les jeux de données sources (fichiers CSV).
- `rapport_ventes.pbix` : Le fichier de projet Power BI principal (si disponible).
- `README.md` : Documentation du projet.

## 🛠️ Technologies & Outils
- **Power BI Desktop** : Pour la visualisation et le reporting.
- **Power Query** : Pour l'extraction, la transformation et le chargement des données (ETL).
- **DAX (Data Analysis Expressions)** : Pour la création de mesures calculées (KPIs).
- **GitHub** : Pour le versioning du projet.

## 📈 Indicateurs Clés (KPIs) analysés
Le rapport met en avant plusieurs mesures stratégiques :
- **Chiffre d'Affaires Global** : Somme totale des ventes.
- **Volume des Ventes** : Nombre d'unités vendues par période.
- **Analyse Temporelle** : Évolution des ventes par mois et par année.
- **Performance Produits** : Top 5 des produits les plus vendus.
- **Analyse Géographique** : Répartition des ventes par région ou secteur.
